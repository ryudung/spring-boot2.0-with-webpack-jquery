# 프론트 개발 세팅.    

### 프론트 설치법.    

1. npm 설치    
    
    
```bash
 npm install  
```    

    
### 프론트 개발 방법.

1. webpack build (최초 1회 실행)

```bash
 npm run build  
```
    
- base.ftl 내부에 script가 생성된 걸 확인한다.
- resources/static/js/ 에 build.js가 생성된 걸 확인한다.
    
2. spring boot run.    
    
3. webpack dev server run.    
    
```bash
 npm run dev  
```
    
4. localhost:3000 접속.    
    
5. 소스코드 변경시, hot-reload 기능으로 개발시작.
